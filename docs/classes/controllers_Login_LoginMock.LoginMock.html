    Class LoginMock

    🧪 LoginMock

    Implementación de LoginInterface<MockUser> para simular autenticación en entornos de desarrollo o testing sin conexión a una base de datos real. Utiliza valores predefinidos desde AUTH_CONFIG_MOCK.

    Se puede utilizar como fallback o mock durante pruebas unitarias.

      👁️ onAuthStateChanged

      Simula restauración del usuario autenticado leyendo el estado desde localStorage. El valor se guarda bajo la clave AUTH_CONFIG_MOCK.STORAGE_KEY.

      Parameters

      • callback: (user: null | MockUser) => void

        Función invocada con el usuario activo o null.

      Returns () => void

      Función vacía para cumplir con el patrón de suscripción.
