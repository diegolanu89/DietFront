    Class LoginFirebase

    🔐 LoginFirebase

    Implementación concreta del patrón Adapter que utiliza Firebase Authentication para gestionar el inicio de sesión, cierre de sesión y observación del estado del usuario.

    Esta clase implementa la interfaz LoginInterface<AppUser>, lo cual permite que sea intercambiable por otras implementaciones como Mongo o Mock.
