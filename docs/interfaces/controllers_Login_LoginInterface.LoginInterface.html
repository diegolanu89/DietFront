    Interface LoginInterface<TUser>

    🔄 LoginInterface

    Interfaz genérica que define el contrato que deben cumplir todos los adaptadores de autenticación. Permite abstraer la lógica de login/logout y observar cambios de sesión, independientemente del proveedor (Firebase, MongoDB, mock, etc.).

    Esta interfaz asegura consistencia en el uso del contexto de autenticación en toda la aplicación.

    interface LoginInterface<TUser> {
        login(email: string, password: string): Promise<TUser>;
        logout(): Promise<void>;
        onAuthStateChanged(callback: (user: null | TUser) => void): () => void;
        register(email: string, password: string, nombre?: string): Promise<TUser>;
    }

      Tipo del objeto usuario que será retornado al loguear y observar sesión.
